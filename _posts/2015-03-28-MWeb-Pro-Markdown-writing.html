MWeb - Pro Markdown writing, note taking and static blog generator App

Pro Markdown writing

  • Mini UI,beautiful Markdown syntax highlight,optimized lists indent, seven editor themes. Support official Markdown syntax and massive Markdown extensions:
    • Support Table
    • Support TOC（Table Of Content）
    • Support MathJax
    • Support Fenced code block
    • Support Task lists, - + space + [ ]or - + space + [x]
    • Support Sequence & Flow chart
    • Support Strikethrough、Underline、Superscript、Autolinks、Footnote and more.
  • Support six live preview themes. Easy to custom preview theme. All themes can export to HTML or PDF. To open live preview window, please press CMD + R.
  • Support Outline view.

Design with two patterns

  • External Document pattern: This pattern design for create, open external Markdown document, it just like TextEdit.
  • Documents Library pattern: This pattern design for note taking and static blog generator. You can organize documents with categories,drag & drop or paste photos to the editor and display. CMD + V paste to JPG pictrue, CMD + Shift + V paste to PNG pictrue.

CMD + N and CMD + O to create,open external Markdown document.
CMD + Shift + N to create MWeb Library document. CMD + L to open MWeb Documents Library.

Documents Library pattern for note taking

Documents Library pattern use categories tree to organize documents, support drag & drop or paste insert pictrue and display immediately. Support import external Markdown or txt document to the library, export a category or documents to HTML, PDF or Markdown.

More info, please check this post: Introduction MWeb Documents Library

Documents Library pattern for static blog generator

You can build static blog One Click! Now has two blog themes, it's easy to custom you owner blog theme. you can add AddThis and Disqus code for share and comment support to you blog. You can enable MathJax and Sequence & Flow chart support in category setting.
